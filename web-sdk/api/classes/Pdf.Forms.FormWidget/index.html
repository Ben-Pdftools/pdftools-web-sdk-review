noPrint​

• noPrint: boolean = true

The annotation's visibility when printing

If true, the annotation is not printed, regardless of whether it is displayed on the screen. If false, the annotation visible when the page is printed. This can be useful, for example, for annotations representing interactive buttons, which would serve no meaningful purpose on the printed page.

Default Value

true
